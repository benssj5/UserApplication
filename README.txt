Example Application JAVA Springboot with : 

- Spring MVC
- H2 Database
- Thymeleaf

Access DB H2 : http://localhost:8080/h2-console

Entity : User (id, name, lastname, age)

This application allow on a One-Web-Page to show :
- the list of all users
- Add an user
- Edit an user
- Delete an user

Need to run the application et launch http://localhost:8080
